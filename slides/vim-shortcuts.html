Vim Shortcuts

Add configs into your ~/.vimrc


" Shortcuts
" ------------------
set pastetoggle=<F2>
set pastetoggle=<F2>
"let mapleader=","
"set timeout timeoutlen=1500
map <F3> :NERDTreeToggle <cr>
map <F2> :tabnext <cr>
map <F5> :w <cr>
" use pbcopy in mac
map <F6> !pbcopy <cr> u <cr>

Those are some examples of my personal preferences, if you have a touchbar instead of Fn keys you might want to re-configure that
