Vim NerdCommenter

Install plugin


cd ~/.vim/bundle
git clone https://github.com/scrooloose/nerdcommenter.git

Add configs into your ~/.vimrc


" Plugin: NerdCommenter
" https://github.com/scrooloose/nerdcommenter
" ------------------
" for nerd commenter
filetype plugin on

Visit the oficial website for the latest instructions https://github.com/scrooloose/nerdcommenter
