Vim Backups

Create a backup file


mkdir ~/.vimbackuptemp

Add configs into your ~/.vimrc


" Backups
" ------------------
" Directory backup
" http://stackoverflow.com/questions/4824188/git-ignore-vim-temporary-files
set backupdir=~/.vimbackuptemp
set directory=~/.vimbackuptemp

Otherwise you will end up with tons of swap files all over the place
