Vim NerdTree

Install plugin


git clone https://github.com/scrooloose/nerdtree.git ~/.vim/bundle/nerdtree

Add configs into your ~/.vimrc


" Plugin: NerdTree
" ------------------
" https://github.com/scrooloose/nerdtree
" nerd tree icon
let g:NERDTreeNodeDelimiter = "\u00a0"

Otherwise you will end up with tons of swap files all over the place
